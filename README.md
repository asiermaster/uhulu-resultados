# uhulu-resultados

Panel de clasificaciones y estadísticas para la quiniela ciclista de la Peña Astopotro / Uhulu Bike tropela.

### 🏠 Páginas

- **index.html** — Medalleros por año/década, puntos, top 10, "cuanto peor mejor"
- **participantes.html** — Perfiles de participantes con gráfico de evolución histórica

### 🛠️ Stack

HTML5 + CSS3 + JavaScript vanilla · Bootstrap 5.1.3 · Bootstrap-Table 1.22.6 · Chart.js · jQuery

### 📡 Despliegue

Hosteado en [GitHub Pages](https://asiermaster.github.io/uhulu-resultados/). Sin backend — los datos son arrays JS estáticos en `data/`.

### 📁 Estructura

```
css/           → Estilos (sidebar, medallas, participantes)
js/            → Lógica de cada página
data/          → Datos estáticos (medalleros, puntos, clasificaciones)
images/        → Avatares de participantes
```
